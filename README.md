# PseudoCode Studio 🚀

PseudoCode Studio is a desktop-like algorithmic workbench designed to transform natural language pseudocode into production-ready code. It provides a structured workflow for problem definition, logic validation, security auditing, and multi-language translation.

## ✨ Key Features

- **Algorithmic Workflow**: A guided 4-step process:
  1. **Problem Definition**: Describe your challenge in plain English.
  2. **Logic Design**: Write pseudocode with real-time stats.
  3. **Security Audit**: Automatically scan for logical vulnerabilities.
  4. **Translation**: Convert logic into TypeScript, Python, JavaScript, Rust, Go, or SQL.
- **Desktop Environment**: A multi-window interface with maximizing/minimizing capabilities.
- **Visual Indicators**:
  - **Workflow Stepper**: Track your progress through the development lifecycle.
  - **Editor Stats**: Real-time line and word counts for both problem and logic editors.
  - **Logic Health Badge**: Instant feedback on the security status of your algorithms.
- **Refinement Analysis**: AI-powered insights that suggest clarifying questions and edge cases for your problem statements.
- **Flowchart Generation**: Automatically visualize your logic using Mermaid.js.

## 🛠️ Tech Stack

- **Frontend**: React 18+ with TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Diagrams**: Mermaid.js
- **Markdown**: React Markdown

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📖 How to Use

1. **Define the Problem**: Open the "Problem" window and describe what you want to build.
2. **Analyze**: Use the "Refine" button to get AI suggestions on edge cases.
3. **Write Pseudocode**: Open the "Editor" and draft your logic.
4. **Validate & Audit**: Use the "Validate" and "Audit" buttons to ensure your logic is sound and secure.
5. **Process**: Click "Process" to translate your pseudocode into your target programming language.
6. **Export**: Copy the generated code or download your logic as a file.
